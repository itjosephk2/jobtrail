- Python (Django)
- HTML/CSS/JavaScript

- Heroku (deployment)

## Getting Started

If you would like to run the project locally:

1. Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3. Run the development server:
    ```bash
    python manage.py runserver
    ```
